# nrubik2
An N-Curses Based, Virtual Rubik's Cube (complete)

This new version was inspired by the original **nrubik**.

![Nrubik2](nrubik2.jpg?raw=true)

 - Full movements implemented
 - Full undo/redo support
 - Stoppable timer
 - Two layouts
 - Three challenges
